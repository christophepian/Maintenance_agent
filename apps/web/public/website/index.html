01 — Maintenance

From tenant message to closed job — without lifting a finger.

When a tenant reports an issue, Propfolio classifies it, routes it to the right contractor, keeps everyone updated, and closes the invoice — automatically. You step in only when a decision actually needs you.

✓
Tenants report via a clean portal — no WhatsApp
Structured intake means contractors always have the full context before they show up.
✓
Set approval thresholds per building
Routine repairs go straight through. Big jobs wait for your sign-off. You define the rules once.
✓
Every invoice captured and posted to your ledger
Photograph or upload the contractor's bill. Propfolio reads it, drafts the entry, and books it.

Built for Switzerland

Swiss compliance, standard — zero configuration required.

Not a generic platform adapted for Switzerland. Every rule, format, and legal notice is built in from day one.

🏦
SIX QR-Bill (ISO 20022)
Every rent demand generates a compliant QR payment slip. Tenants pay directly from the portal.
📜
CO 259a / 269d tenancy law
Live SNB reference rate ingestion. Rent adjustments flagged and notices drafted automatically.
🗂️
PAIF depreciation standards
Swiss asset taxonomy, residual value calculation, and per-intervention deduction tracking.
🌐
FR / DE / IT — all cantons
Canton-specific lease templates, multilingual tenant portal, and bilingual legal term matching.
Why Propfolio

What makes it different.

Most property tools are either too simple for professional managers or too complex for small landlords. We built both, on the same platform.

01
One source of truth
Maintenance events, lease data, invoices, and legal cases all live in the same system and cross-reference each other. No more spreadsheet bridges between tools.
02
End-to-end automation
From tenant request to closed invoice, every step flows automatically. No handoffs between apps, no copy-pasting, no chasing — the platform runs the loop.
03
All four parties, one platform
Manager, owner, tenant, and contractor each have a role-scoped interface. Communication flows through the platform, not via email chains and WhatsApp.
04
Real-time financial cockpit
Live ledger, per-building P&L, and portfolio NOI — updated the moment each transaction posts. Owners stop waiting for the accountant's file in February.
05
Scales from 1 to 1,000 units
The same backend powers the professional manager view and a simplified private landlord mode (Phase 2). No separate product, no data migration.
06
AI-augmented by design
Triage, classification, and legal analysis already use rule-based AI. The architecture is built to add LLM-based triage and self-service resolution as the next layer.
